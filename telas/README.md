# Telas implementadas

Esta pasta deve conter as quatro telas referente as entrega dos protótipos de tela.

Indique quais telas você escolheu e faça um breve descrição de cada uma delas. 

## Exemplo

- [Tela de início](#)
  - Tela responsável por apresentar a página inicial da loja virtual Manicure Store. Nela são exibidos os principais destaques do site, como produtos e cursos em evidência, além de uma seção principal de apresentação da marca com botões de navegação para outras páginas do sistema. A tela tem como objetivo recepcionar o usuário e direcioná-lo para as principais funcionalidades da loja

- [Tela de Produtos](#)
  Tela responsável por apresentar os produtos disponíveis na loja virtual. Nela são exibidos cards contendo imagem ilustrativa, nome do produto, descrição, preço e botão de compra. O objetivo dessa tela é permitir que o usuário visualize os itens comercializados de forma organizada, simples e intuitiva.
